# Phase Banners

## When to Display
- At the start of every phase
- On every session resume
- Whenever the phase or unit changes
- Always display BEFORE doing any other work

## Standard Phase Banner (Phases 1–4)

```
╔══════════════════════════════════════════════╗
║  📍 PHASE X: <PHASE NAME>                    ║
║  Status: <in-progress | awaiting approval>   ║
╚══════════════════════════════════════════════╝
```

## Execution Banner (Phase 5)

```
╔══════════════════════════════════════════════╗
║  📍 PHASE 5: EXECUTION                       ║
║  Version: vX | Unit Y: <unit name>           ║
║  Status: <implementing | testing | reviewing ║
╚══════════════════════════════════════════════╝
```
